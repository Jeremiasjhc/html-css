Em um site nós podemos usar listas

Em HTML, temos Listas ordenadas, não ordenadas, e lista de definições

  1. LISTAS ORDENADAS
    • As listas ordenadas são usadas para ordenar ou organizar coisas
    • Exemplo:1,2,3; A,B,C etc
    • Tambem podemos trocar os formatos das ordem, para isso usamos o type, e o startype.

  2. LISTAS NÃO ORDENADAS
    1. As listas não ordenadas são usadas para mostrar ao leitor que tem uma listas de coisas á serem feitas
    2. De igual modo podemos trocar os formatos das ordens usando o type.

  3. LISTAS DE DEFINIÇÕES
    • As listas de definições servem para mostrar a descrição do conteúdo da lista
    • Na lista de definições temos o dt_ para mostrar o conteúdo e o dd_ para a descrição do conteúdo.
